在mac中创建脚本文件
2021-01-11|技术问题

我在使用hexo建立自己的博客的时候，发现hexo 需要部署自己写的文章的时候的过程特别麻烦需要

  1. hexo clean
  2. hexo g
  3. hexo d
  4. hexo algolia

如果需要手动输的话，不仅很烦，还需要等待之前的执行完毕

因此需要创建一个脚本文件（所有在终端执行的多行指令都可以写成脚本文件直接整个执行）

过程大致如下

  1. 进入终端

  2. 输入touch fileName.sh创建一个shell文件

  3. 赋予该文件可执行权限chmod 755 fileName.sh(这一步非常重要，没有这一步文件是没法执行的)

  4. 编辑shell文件，vi fileName.sh

  5. 执行该文件./fileName.sh即可
