变量

js的变量是松散类型的，其可以存储任意类型的值

但是在使用过程中，最好不要改变变量的类型

变量声明

使用var声明的变量为局部变量，只会在其所在的代码块中存在，一旦出了最近的代码块，该变量会被销毁

但是也可以直接声明变量而不使用var,此时声明的是全局变量

可以使用以下方法声明多个变量

数据类型

使用typeof操作符可以获得表示该变量的类型的字符串

数据类型主要有以下几种

类型 表示 例子
Undefined 只声明了的变量,未被初始化 var message;
Null 空对象指针 var message=null;
Boolean 布尔值(true/false)，只能小写 var have=false;
Number 数值类型 var num=10;
String 字符串类型(可用‘ ’或者“ ”) var str="hello world";
Object 对象类型 var o=new Object();

Number可以表示十进制，八进制和十六进制数。八进制需要0开头，而十六进制需要0x开头

注意在js中的浮点数存在误差，所以不要使用浮点数直接进行比较

js中的数值存在范围，如果超过范围会变成INFINITY,可以通过函数isFinite()来判断是否是无限大

isNaN可以用来判断一个变量是否是数或者能否转换为数

对可以转换为Number的变量使用Number()可以将其转换为Number

String类型的变量可以通过+/-进行一定的操作，且几乎任意变量都有toString()方法将其变成字符串形式（一般需要重写）

相等操作符

js中有两种操作符，一种是相等/不相等，另一种是全等/不全等

相等判断时会先将能转换的先转换成同一个再进行比较

全等不会进行转换，如果类型不同一定不全等

变量，作用域及内存

基本类型和引用类型

js的变量可以划分为两类，一种是基本类型，另一种是引用类型

对于引用数据类型，可以为其添加属性和方法（类似于类）

对于基本数据类型，复制就是创建为变量赋一个等同与被复制值的新值

而对于引用数据类型，则是会指向同一引用副本，类似于指针

在js中均为传递参数，而不是传递引用，但是对于值的访问有按值和按引用进行两种。因此可以向函数传递一个引用类型变量，同样能够达到传递引用的效果（但不是传递引用）

看以下经典例子

要判断一个变量是否为引用数据类型，只需要使用typeof操作符即可。但是这种方法只能返回是否是一个引用数据类型，不能判断是哪种引用数据类型

因此可以使用instanceof（二元操作符）

对所有基本数据类型使用instanceof都会返回false

作用域

注意js与其他语言不同，没有块作用域

对于C++语言，任意代码块都是一个作用域

但是对于js来说，只有函数才有作用域，其他是没有的

如果不使用var进行声明，则是全局变量，能够在任意函数进行使用

垃圾收集

js的垃圾收集有两种方式：

  1. 标记清除

    通过给变量加标记实现，当变量离开环境时，即不可能在被使用时，则打上标记，等待垃圾收集器进行收集工作。

  2. 引用计数（使用的较少）

    即每次被引用就在计数器中递增引用次数，当引用计数为0的时候进行垃圾回收♻️。

    但是这样有一个严重的问题：可能会有循环引用

    这样在删除的时候就形成一个循环死锁无法删除，因此之后不经常采用。

js现在自己有使用第一种方式的垃圾回收♻️机制，因此不用特别在意。但是由于网页允许的最大内存比在计算机中的少，以防止网络崩溃。因此最好在确定不需要使用引用的时候，手动写上A=null;即可。

引用类型

在js中，一般将引用类型的实例称为对象

创建对象的方法一般有两种：

  1. 使用new

  2. 对象字面量方法

一般最常使用的是对象字面量方法，特别是多属性的情况

Array类型

创建数组的方式有两种

  1. 使用构造函数进行构造

  2. 使用数组字面量表示法

而在读取值的时候只需要按照正常读取数组的形式即可。

可以通过length属性访问数组的长度

Array中的方法：

方法 效果
push() 将值加入数组的末尾
pop() 移除最后一项并返回被移除的值
shift() 移除第一项并取得被移除的值
unshift() 将值加入数组的最前面
reverse() 将数组顺序颠倒
sort(compare) 将数组进行排序（可以自己写比较函数）
concat() 基于传入的值叉u给你讲爱你一个新的数组
slice(i,j) 使用i,j之间的部分创建一个新数组
splice() 可以删除，插入，添加
indexOf() 从开头开始查找
lastIndexOf() 从末尾开始查找

特别注意splice,splice中可以接受三个参数分别为

  1. 起始位置
  2. 删除位数
  3. 添加项

使用splice可以很容易的操作数组中的内容

数组迭代

方法 作用
every() 对数组中的每一项调用给定函数，若全部true则返回true
filter() 对数组中的每一项调用给定函数，返回boolean数组
forEach() 对数组中的每一项调用给定函数
map() 对数组中的每一项调用给定函数，返回返回值数组
some() 对数组中的每一项调用给定函数，只要有一项为true则返回true

以上函数都接受一个function类型的参数，可以是已经写好的也可以是临时定义的。一般为临时定义的，因为有参数的限制

函数类型

js中的函数有一个很重要的概念就是：

函数是对象，函数名是指针

函数没有重载（之前已经说过）

由于js的函数可以没有名字，如果有名字那么在递归的时候调用名字即可，但是对于没有名字的函数则应该用一下方式递归

函数具有两个属性：

  1. length即希望返回的参数个数
  2. prototype函数的原型（待补充）
