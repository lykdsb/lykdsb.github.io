解决java写文件失败
2020-09-18|技术总结

问题

今天在尝试使用java中的读文件方法PrintWriter的时候，发现在程序中写入文件之后，程序结束后文件中并没有我输入的东西

在执行上述代码之后，该文件中并没有显示出Greeting字符串

解决

在查阅资料后发现，在使用完PrintWriter之后，是一定需要将该对象关闭的，否则就会出现文件被占用，一直无法读取的状态

将代码更改为以下形式能够正确得执行了

应当养成一种习惯，在用完流或与流类似的对象后一定要将该流关闭

另附

public void close()

Closes the stream and releases any system resources associated with it. Closing a previously closed stream has no effect.
